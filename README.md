# 🌿 MentAura

> **Revolutionizing mental wellness through Agentic AI, community support, and privacy-first innovation.**

MentAura is an **AI-powered mental wellness ecosystem** designed to provide personalized emotional support through a collaborative **5-agent AI architecture**. The platform combines intelligent conversations, community engagement, multilingual accessibility, and blockchain-enabled privacy to create a safe and inclusive environment for mental well-being.

---

## ✨ Features

* 🤖 **5-Agent AI Chat System**

  * Orchestrator
  * Emotion Bot
  * Wellness Strategy Agent
  * Context Memory Agent
  * Response Generator

* 🧠 **Emotion-Aware Conversations**

  * Detects emotional states and provides empathetic responses.

* 🌍 **Multilingual Support**

  * Makes mental wellness resources accessible to diverse users.

* 👥 **Community Forum**

  * Enables users to connect, share experiences, and support each other.

* 📚 **Resource Hub**

  * Curated articles and educational content for mental well-being.

* 🔐 **Multiple Authentication Systems**

  * Secure user login and session management.

* ⛓️ **Blockchain-Based Privacy**

  * MetaMask integration for decentralized identity and secure file handling.

* 💙 **Personalized Support**

  * Maintains contextual memory for meaningful and continuous conversations.

## 🎯 Problem Statement

Mental health support remains inaccessible, expensive, and often lacks personalization. Existing chatbot systems provide generic responses and fail to maintain contextual understanding.

**MentAura addresses this challenge through a multi-agent AI architecture that delivers empathetic, personalized, and privacy-first emotional support while fostering a supportive community.**
